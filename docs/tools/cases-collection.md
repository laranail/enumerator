# Cases collection

`YourEnum::collect()` returns a `Support\CasesCollection` that extends Laravel's `Illuminate\Support\Collection` with enum-aware helpers: `names()`, `values()`, `labels()`, `whereName()`, `whereValue()`, `whereMeta()`, `only()`, `except()`, `keyByName()`, `keyByValue()`, `pluck()`, `toRichArray()`.
