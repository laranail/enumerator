# Tailwind config

Add the Tailwind view bundle path to your `tailwind.config.js`:

```js
module.exports = {
  content: [
    './resources/views/**/*.blade.php',
    './vendor/laranail/enumerator/resources/views/components/tailwind/**/*.blade.php',
  ],
};
```
